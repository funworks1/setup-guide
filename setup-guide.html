Office（Word / Excel / PowerPoint）

インストール手順
office.com を開く
  1. office.com にアクセスし、会社のMicrosoftアカウントでサインイン
  2. 左上の アプリ起動ツール（⠿） をクリック →「その他のアプリ」を選択
  3. 「↓ アプリをインストール」→「Microsoft 365 Apps」をクリック
  4. 「↓ アプリとデバイス」→「Office のインストール」をクリックしてダウンロード
  5. ダウンロードしたファイルを実行してインストール完了
インストールされる主なアプリ
  • Word　― 文書作成
  • Excel　― 表計算・データ管理
  • PowerPoint　― プレゼンテーション
  • Outlook (Classic)　― メール・予定表（手順5で設定）
  • OneDrive　― クラウドストレージ（手順9で設定）

Outlook（Classic推奨）・メール設定

Outlook の起動確認

Office（手順1）をインストールするとOutlookが含まれます。スタートメニューから「Outlook (Classic)」を起動してください。

📌 現在は Outlook (Classic) を推奨しています。
当社のメール設定はPOP接続を使用しており、Outlook (New) はPOP接続に正式対応していないため、会社メールアドレスが正常に受信できません。

⚠️ PCに Outlook (New) がインストールされている場合はアンインストールしてください。
※ ご自身のメールアドレスで使用中の方はそのままで構いませんが、会社メールアドレスの設定は必ずOutlook (Classic) で行ってください。
メールアカウントの設定
📌 IFA基本情報にメールアドレスとパスワードが記載されています。事前にご確認ください。

▶ 手順1でインストールした方：起動するとすぐにアカウント設定画面が表示されます。そのまま下記の手順で進んでください。

▶ もともとClassicが入っていた方：「ファイル」→「アカウントの追加」から設定画面を開いてください。

  1. メールアドレスを入力
  2. 「詳細オプション」をクリック →「自分で自分のアカウントを手動で設定」にチェックを入れて「接続」
  3. 「POP」を選択
  4. パスワードを入力して「接続」
  5. 「インターネット セキュリティの警告」が表示されたら「はい」をクリックして完了

Slack

インストール
Slackダウンロードページ
  1. 「ダウンロード（64ビット）」をクリックしてインストーラーを保存
  2. ダウンロードした「SlackSetup.exe」を起動
  3. インストールが自動で進み、完了するとSlackが自動的に起動する
  4. 「Sign in to Slack」の画面が表示されたらサインインへ進む
サインイン
  1. アプリを起動し、「ワークスペースにサインイン」をクリック
  2. 会社メールアドレスを入力（IFA基本情報に記載のOutlook (Classic) メールアドレス）
  3. 会社メールアドレス宛に届いた認証コードを入力してログイン
⚠️ 認証コードが会社メールアドレスに届くため、先に手順5（Outlook (Classic)）のメール設定を完了させてからサインインしてください。

支店プリンタドライバー

🏢 支店プリンターの設定（支店に行ったときに対応）

支店プリンターの設定は、実際に支店に行ったタイミングで行ってください。

  1. 支店のプリンター機種名を確認（例：IMC2500）
  2. ブラウザで「機種名　ドライバー」と検索（例：IMC2500 ドライバー）
  3. メーカー公式サイトからドライバーをダウンロードしてインストール
  4. 手順通り進め、「ネットワークから検索」で支店のプリンターが表示されたらそのまま設定を完了
⚠️ カラーモードの設定を忘れずに。
「設定」→「Bluetoothとデバイス」→ 該当プリンターを選択 →「印刷設定」→ カラーモードを「白黒」に変更
🏠 自宅プリンターの接続（自宅で作業する場合）

自宅では、ご自身のプリンターをPCに接続してご利用ください。プリンター付属の手順書またはメーカーサイトを参照してセットアップしてください。

OneDrive 同期設定（会社アカウント）

OneDriveの起動

Windows 11にはOneDriveが標準搭載されています。タスクバーのOneDriveアイコン（雲マーク）をクリックして起動してください。

会社アカウントでサインイン
  1. OneDriveアイコンを右クリック →「設定」→「アカウント」→「アカウントの追加」
  2. 会社のMicrosoftアカウント（メールアドレス）を入力
  3. IFA基本情報に記載のMicrosoftアカウントのパスワードを入力してサインイン
  4. 同期フォルダの選択画面では、すべてのチェックをOFFにして次へ進む
📌 パスワードを変更した場合は、必ずIFA基本情報に保存してください。
❓ パスワードがわからなくなった場合は、Slackで @事務員 にご連絡ください。
会社ワンドライブからの同期設定

会社会社ワンドライブのURLを後ほどお送りします。URLが届いたら以下の手順で同期してください。

  1. 送付された会社ワンドライブのURLをブラウザで開く
  2. ツールバーに「同期」が表示されていればそのままクリック。見当たらない場合は「…」（三点リーダー）の中にあります
  3. 「OneDriveを開く」または「許可」のダイアログが表示されたら許可する
  4. 完了するとエクスプローラーにビルマークの会社フォルダが表示される
⚠️ 【重要】データ保存に関するルール

業務で使用するファイルは、必ずビルマーク（会社ワンドライブ）のご自身のフルネームフォルダ内に保存・管理してください。

以下には保存しないでください：
・デスクトップへの保存
・ご自宅のPC内（ローカル）への保存
・☁️ 雲マーク（個人用OneDrive）への業務データの保存

また、個人情報を含むファイルを扱う場合は、一時保管のみとし、作業完了後はすみやかに削除してください。
🗑️ ゴミ箱の中も毎日削除してください。ファイルを削除してもゴミ箱に残っている間はデータが消えていません。

PDFアプリの既定設定

設定手順
  1. 「スタート」→「設定」（⚙️）を開く
  2. 「アプリ」→「既定のアプリ」をクリック
  3. 「Adobe Acrobat」を検索して選択

確認方法

Edge IEモード設定

設定手順
  1. Microsoft Edgeを起動
  2. 右上の「…」（三点メニュー）→「設定」をクリック
  3. 左メニューの「既定のブラウザー」をクリック
  4. 「Internet Explorer モード (IE モード) でサイトの再読み込みを許可する」を「許可」に変更
  5. 「再起動」ボタンをクリックしてEdgeを再起動
IEモードが必要なサイトは、Edgeで開いた後「…」→「IEモードで再度読み込む」から切り替えられます。

ウイルス対策・セキュリティ設定 ⚠️

⚠️ PCのセキュリティはご自身の責任で管理してください。不安な方はSlackでご相談ください。会社からマカフィーを提供することも可能です。
セキュリティの基本ルール
  • ウイルス対策ソフトを常に有効にする
    → 設定 ＞ プライバシーとセキュリティ ＞ Windowsセキュリティ ＞ オンを確認
  • Windows Updateは放置せず必ず実施する
    → 設定 ＞ Windows Update ＞ 更新プログラムの確認
  • 怪しいメールや心当たりのない添付ファイルは開かない
  • 業務用PCはご本人のみが使用する
    （ご家族など第三者が使用すると、意図せず不審なサイトへのアクセスやソフトのインストールが行われる可能性があるため）
BitLockerの設定

PCのドライブ全体を暗号化する機能です。紛失・盗難時に第三者へのデータ流出を防ぎます。必ず設定してください。

  • 設定 ＞ プライバシーとセキュリティ ＞ デバイスの暗号化 ＞ オンにする　※ BitLockerドライブ暗号化と表示される場合は「有効にする」をクリック（Windows 11 Pro）
